products api documentation

Routes

All HTTP methods are supported. You can use http or https for your requests.

Get products

  • Get products:/products (Get)
  • Search products by name or info:/products/?s=key word (Get)
  • Get 10 products by page number:/products/?page=number (Get)
  • Get all products by category :/products/cat/:categoryName (Get)
    • Cars \ Bullets \ Rubiks Cube \ Rings \ Wooden blocks \ circles \ home \ animal \ lgo \
  • Price range:/products/prices/?min=10&max=40 (Get)

Add products

Get products - (only for register user - must send user's auth-token through headers):/products (Post)
  • name (required)
  • info (required)
  • price (required)
  • mkt (required)
  • category (required)
  • img_url

Edit products

Edit products - (only for register user - must send user's auth-token through headers):/products/:_id (products) (PUT)
  • name (required)
  • info (required)
  • mkt (required)
  • price (required)
  • category (required)
  • img_url

Delete products

Delete products - (only for register user - must send user's auth-token through headers): /products/:_id (products) (DELETE)
